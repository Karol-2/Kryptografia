
KOD NAPISANY W PYTHON 3.11.2

Kod dzieli domyślnie teskt na linijki po 64 znaki
jeżeli wiersz jest krótszy niż 64, dopisywane są "x" w odpowiedniej ilości

Polecenia: 

przygotowanie tekstu do przykładu działania
python xor.py -p

szyfrowanie
python xor.py -e

kryptoanaliza wyłącznie w oparciu o kryptogram
python xor.py -k
